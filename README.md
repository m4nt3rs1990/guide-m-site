# guide-m-site

本仓库用于归档和发布多个独立 HTML 页面。这些页面以静态方式存放，不针对任何单个域名或具体网站，仅作为内容存档与展示用途。

## 项目简介

guide-m-site 是一个通用型 HTML 页面仓库。你可以在此找到一系列独立页面，每个页面均以自包含的 HTML 文件形式存在，方便直接查看或部署。

## 目录说明

仓库根目录下直接存放 HTML 文件。部分页面可能附带相关资源文件夹（如 CSS、JS、图片等），文件夹命名与 HTML 文件名称对应。

```
/
├── index.html
├── page-example.html
├── page-example/
│   ├── style.css
│   └── script.js
└── ...
```

## 页面归档说明

- 每个 HTML 文件均为独立页面，可直接在浏览器中打开。
- 页面之间无强关联，可单独使用或部署。
- 所有页面均为静态内容，不依赖后端服务。
- 页面内容可能包含示例、说明或演示性信息。

## 维护说明

- 本仓库不定期更新，新增或修改页面。
